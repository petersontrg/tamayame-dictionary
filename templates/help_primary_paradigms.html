Primary Paradigm Classes in Tamayame

In Tamayame, transitive verbs are divided into four Primary Paradigm Classes (A–D), based on which sets of Person, Role, Mood, Polarity prefix allomorphs (PRMPs) they use. Each class corresponds to a fixed combination of three partial paradigms—one for 1st person, 2nd person, and 3rd person. These patterns reflect person hierarchy and inverse-like alignments common in many Indigenous languages of North America.

Class overview

Each class bundles three partial paradigms (1st, 2nd, 3rd person):

Class Partial paradigms Interpretation
A , , Direct/canonical alignment across persons.
B , , Alternate 1st/2nd person forms; direct 3rd.
C , , Inverse-leaning: marked patterns for 3rd acting on 1st/2nd.
D , , Hybrid: alternate 1st, regular 2nd, inverse-leaning 3rd.
Important: The Classes A–D apply to singular objects. When the object is dual or plural, all transitive verbs take Class A regardless of their singular class.

A typological perspective. These combinations mirror widely attested person/animacy hierarchy effects: 1st > 2nd > 3rd, where 3→1/2 configurations are comparatively marked in many languages.

Why it matters

  • Prediction: Knowing a verb’s class tells you which PRMP prefixes it combines with.
  • Learning: Helps learners connect forms across persons/objects more quickly.
  • Searching: The dictionary can surface all compatible prefixes for the verb root’s class.

On each dictionary entry page, the Primary Paradigm Class automatically expands to show the full set of pronominal prefixes that work with that verb root.

Terminology

PRMP (Person, Role, Mood, Polarity)
Sets of person/number-marking prefixes on the verb. They can vary by context and verb class.
Partial paradigm (e.g., 1a, 2b, 3a)
A named set of PRMP allomorphs for a given person that repeats across verbs and classes.
Primary Paradigm Class (A–D)
The bundle of the three partial paradigms a verb uses (1st, 2nd, 3rd person). For dual/plural objects, the system defaults to Class A.
