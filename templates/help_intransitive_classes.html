Intransitive Verb Classes in Tamayame

Intransitive verbs in Tamayame are grouped into formal classes based on the pronominal prefixes (PRMPs) they take. These classes reflect which person/number combinations the verb occurs with in real usage, and are essential for capturing affix distribution, especially for verbs that exhibit number-marked patterns.

Class Codes

Most intransitive verb classes are represented by a two-part code: X–Y. The first number (X) identifies the affix set used for:

  • 3rd person subject (affix 121–126)
  • 1st person negative (affix 102)
  • 2nd person negative hortative (affix 115)
The second number (Y) identifies the set of affixes used for 1st and 2nd person forms (affixes 101, 103, 104, etc.).

For example, Class 1–3 uses 3rd person affixes from group 1, and 1st/2nd person affixes from group 3. These class labels appear on entry pages to indicate how the verb behaves morphologically.

Number-Based Usage

Each verb class is used in a specific number context — singular, dual, plural, or a combination:

  • Singular-only: 1–1, 1–3, 4–5, 6–3, 7–13, etc.
  • Dual-only: 2–24, 10–22, 11–26, etc.
  • Plural-only: 1–17, 5–9, 10–21, 14–19, etc.
  • Mixed usage: Class 3–27 occurs in all three numbers; 11–25 is used in both singular and plural.

For many verbs, these class patterns correlate across number. For example:

• 1–1 (singular), 1–18 (dual), 1–17 (plural)

• 5–8 (singular), 5–10 (dual), 5–9 (plural)

• 10–7 (singular), 10–22 (dual), 10–21 (plural)

Semantic and Structural Notes

Verbs are classified based on the actual affixes with which they occur. This means some verbs will be limited to certain forms — for example, only third person or only singular. Class membership does not imply the verb must occur with the full range of affixes in the paradigm.

Some classes (like 7–15) are specialized for kinship, possession, or other semantically limited stems. Others allow benefactive transitivization, meaning they are cross-linked to a transitive verb class.

Database Modeling

In the dictionary, each intransitive class is linked to two affix groups internally and labeled with its class code. Entries show the number usage and the associated TA marker (if applicable), helping identify stems that follow regular number-marked alternation or display exceptional paradigmatic gaps.

Data modeled after Davis (1964), Language of Santa Ana Pueblo. Tamayame orthography has been updated from the original transcription.
